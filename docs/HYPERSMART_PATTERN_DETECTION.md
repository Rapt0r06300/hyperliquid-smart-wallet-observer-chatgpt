# HyperSmart Pattern Detection

Les patterns sont des hypotheses de recherche.

Ils exigent:

- assez d'evidence;
- confidence explicite;
- warnings si PnL concentre;
- message research-only.

Un pattern ne devient jamais un signal d'achat ou vente.
